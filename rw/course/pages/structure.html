Structure

How everything is put together

Scope

Everything in C# is in within a scope. If something is not accessible from the current scope, it cannot be used.
If a scope is within another scope, everything accessible from the containing scope is accessible within the contained scope.
As a general rule, an opening curly brace ({) is the start of a scope, and a closing curly brace (}) is the end of a scope.
 

Code: 

namespace ExampleNamespace { 
    class ExampleClass { 
        string greeting = "Hello"; // these two strings exist within the scope of ExampleClass
        string location = "world"; 
        
        public void SayHello() {
                Console.WriteLine(greeting + " " + location + "!!"); // . . . so they can be used here
        } 
    } 
}
Output:

Hello world!!

Names and Types

Everything in C# has a name and a type.
Everything inside of a class is called a "member". 


string message; // a "string" (text) with the name "message"
int daysInYear = 365; // an "int" (integer) with the name "daysInYear"
List<Lizard> lizards; // a list of lizards named "lizards"

Things also have access modifiers, which denote where else they can be used.
The access modifiers you will run across most often will be "public" and "private".
"public" means that 


class Example {
    string data; // no access modifiers specified, so it is "private"
    public string name;
}
