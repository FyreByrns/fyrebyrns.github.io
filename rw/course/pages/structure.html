How everything is put together

Scope

Everything in C# is in within a scope. If something is not accessible from the current scope, it cannot be used.
If a scope is within another scope, everything accessible from the containing scope is accessible within the contained scope.
As a general rule, an opening curly brace ({) is the start of a scope, and a closing curly brace (}) is the end of a scope.
 
In the code below, bright blue lines are used to show scopes.
 
Code: namespace ExampleNamespace { class ExampleClass { string greeting = "Hello"; These two strings exist within the scope of ExampleClass . . . string location = "world";
public void SayHello() { Console.WriteLine(greeting + " " + location + "!!"); . . . so they can be used here. } } } Output: Hello world!!
