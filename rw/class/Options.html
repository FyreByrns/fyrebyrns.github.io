Options.cs

    object > Options

    Contains game options and controls. Options are loaded from ~/UserData/options.txt

    Constructor

    set resolution to 1
    set soundEffectsVolume to 0.8
    set musicVolume to 0.8
    set arenaMusicVolume to 0.625
    ~/UserData/options.txt exists Load

    Load

    FromString (~/UserData/options.txt)

    FromString

    string s

    set windowed to false
    split s into array with /<optA>/
    for each string in the array split into strings token and values with /<optB>/:
    token = ScreenResolution parse resolution from value 1
    token = InputSetup set controls to ControlSetup.MakeFromString (value 1, value 2)
    token = Windowed set windowed to true
    token = SoundVol parse soundEffectsVolume from value 1
    token = MusicVol parse musicVolume from value 1
    token = ArenaMusicVol parse arenaMusicVolume from value 1
    token = Language parse language from value 1
    token = SaveSlot parse saveSlot from value 1
    token = AllGamePads set allGamePads to true of value 1 is 1, otherwise to false
    token = PlayerToSetInputFor parse playerToSetInputFor from value 1
